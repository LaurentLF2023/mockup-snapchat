Projet développé en React-Native

Création d'une application similaire à Snapchat.

pour lancer le projet vous devez avoir "npm" d'installé ("npm install -g npm"):

- installer l'application "expo go" sur votre téléphone

- depuis la racine du projet installer expo "npm install --global expo-cli"

- taper "npm install" 
pour télécharger tous les paquets nécessaires pour faire tourner l'application.

- lancer un serveur en tapant "expo start --tunnel" à la racine du projet, un QR code devrait apparaitre dans
votre terminal, scanner le et il lancera le projet sur votre mobile
